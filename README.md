# Лидерство для вовлечённости — паспорт программы

Публичная веб-версия паспорта образовательной программы **«Лидерство для вовлечённости: повышаем eNPS команды»** (v01).

## Содержание

- Интенсив 4 недели · 12 часов · смешанный формат
- Целевая аудитория: CEO-1/CEO-2 (топ-менеджеры)
- Цель: рост eNPS с −15 до +10 за 6 месяцев
- Паспорт программы 3.0 (этапы 1–8)

## Локальный просмотр

Откройте `index.html` в браузере — файл самодостаточный, интернет не нужен.
